# 📱 Android - Curiosidades de Tecnologia

Um site informativo e responsivo sobre a história do mascote Android, desenvolvido como portfólio de **Frontend Development**. Demonstra competência em HTML semântico, CSS moderno com animações e boas práticas de acessibilidade.

## 🎯 Objetivo

Criar um site educativo e visualmente atraente que mostre conhecimento em:
- ✨ HTML5 semântico
- 🎨 CSS3 avançado (Variáveis CSS, Flexbox, animações)
- ♿ Acessibilidade Web (WCAG)
- 📱 Design responsivo
- 🚀 Performance e UX

## 🎨 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica com elementos como `<header>`, `<nav>`, `<article>`, `<aside>`, `<footer>`
- **CSS3** - Custom properties, Flexbox, transições suaves, animações CSS
- **Google Fonts** - Tipografia customizada (Bebas Neue)
- **Responsive Design** - Mobile-first approach com breakpoints em 768px e 480px

## ✨ Features

### 🎬 Animações CSS
- Fade-in suave no carregamento da página (header, main, footer)
- Efeito de slide nos headings
- Transições elegantes em hover (links, imagens, botões)
- Staggered animations com `animation-delay`

### ♿ Acessibilidade
- **Contraste melhorado** - Links com background mais escuro (cor3) para melhor visibilidade
- **Focus states** - Navegação e links com outline de 3px ao usar teclado
- **Atributos semânticos** - `aria-label`, `rel="noopener noreferrer"`, `alt` descritivos
- **Scroll behavior** suave

### 📱 Responsividade
```
Desktop: Layout completo com nav horizontal
Tablet (768px): Ajustes de padding e font-size
Mobile (480px): Nav em coluna, imagens otimizadas
```

### 🎯 Boas Práticas
- ✅ Código limpo e bem comentado
- ✅ CSS variables para manutenção fácil
- ✅ Estrutura modular
- ✅ Performance otimizada
- ✅ Imagens com `<picture>` e `srcset` para diferentes resoluções

```

## 🎓 O que Aprendi

Este projeto demonstra:

### HTML
- ✅ Semântica correta (`<header>`, `<nav>`, `<article>`, `<aside>`, `<footer>`)
- ✅ Tags de conteúdo (`<picture>`, `<figure>`, `<abbr>`)
- ✅ Atributos acessíveis (`aria-label`, `alt` descritivos)
- ✅ Links externos seguros (`target="_blank"` + `rel="noopener noreferrer"`)

### CSS
- ✅ Custom Properties (CSS variables) para temas reutilizáveis
- ✅ Animações CSS (`@keyframes`, `animation-delay` para stagger effect)
- ✅ Transições suaves em todos os estados (hover, focus)
- ✅ Flexbox para layouts flexíveis
- ✅ Media queries para design responsivo
- ✅ Box-shadow para profundidade visual
- ✅ Gradientes lineares criando interesse visual

### Acessibilidade (A11y)
- ✅ Contraste de cores WCAG-compliant
- ✅ Focus states visíveis para navegação por teclado
- ✅ Estrutura de heading lógica (h1 → h2)
- ✅ Imagens com alt text descritivo
- ✅ Links com propósito claro

### Performance
- ✅ CSS único (sem frameworks)
- ✅ Imagens otimizadas com srcset
- ✅ Sem JavaScript desnecessário
- ✅ Carregamento progressivo com animações
